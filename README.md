# Black - 专业视频录制应用

一个专为 iPhone 16 Pro (iOS 26) 设计的高质量视频录制应用，支持 4K 120fps 杜比视界录制。

## 功能特性

### 🎬 高质量录制
- **4K 分辨率**: 支持 3840x2160 超高清录制
- **120fps 高帧率**: 提供电影级别的流畅画面
- **杜比视界支持**: 支持 HDR10+ 和杜比视界色彩空间
- **专业级防抖**: 启用电影级防抖功能

### 🖤 极简界面
- **纯黑色界面**: 无论是否深色模式都保持纯黑色背景
- **隐藏状态栏**: 提供完全沉浸式录制体验
- **最小化干扰**: 录制状态指示器透明度降低

### 🔒 智能管理
- **防锁屏**: 录制期间自动防止设备锁屏
- **自动保存**: 锁屏或退出应用时自动保存视频到相册
- **后台处理**: 智能处理应用生命周期事件

## 技术架构

### 核心组件

#### VideoRecordingManager
专业的视频录制管理器，负责：
- AVCaptureSession 配置和管理
- 高质量视频/音频输入设置
- 4K 120fps 格式选择和优化
- 杜比视界色彩空间配置
- 权限管理和错误处理
- 相册保存功能

#### ContentView
主界面视图，实现：
- 纯黑色全屏界面
- 录制状态显示
- 应用生命周期响应
- 防锁屏设置

#### AppStateManager
应用状态管理器，处理：
- 前台/后台状态监听
- 应用激活/非激活事件
- 自动录制控制

### 权限配置

应用需要以下权限：
- **相机访问权限**: 进行视频录制
- **麦克风访问权限**: 录制音频
- **相册访问权限**: 保存录制的视频

## 使用方法

1. **启动应用**: 打开应用后界面显示为纯黑色
2. **自动录制**: 进入界面 0.5 秒后自动开始录制
3. **录制指示**: 左下角显示红点和录制时长（半透明）
4. **保存视频**: 锁屏或退出应用时自动保存到相册
5. **重新录制**: 重新打开应用会开始新的录制

## 系统要求

- **设备**: iPhone 16 Pro 或更新设备
- **系统**: iOS 26 或更高版本
- **存储**: 足够的可用存储空间（4K 视频占用较大）
- **权限**: 相机、麦克风、相册访问权限

## 技术细节

### 视频规格
- **分辨率**: 4K (3840x2160)
- **帧率**: 120fps
- **编码**: H.265/HEVC with Dolby Vision
- **色彩空间**: HLG BT.2020 或 P3-D65
- **音频**: 48kHz 立体声

### 性能优化
- 异步录制处理，避免界面卡顿
- 智能格式选择，确保设备兼容性
- 内存管理优化，支持长时间录制
- 后台任务处理，确保数据安全

### 安全特性
- 录制文件临时存储，保存后自动清理
- 权限检查和用户提示
- 错误处理和恢复机制
- 隐私保护设计
